# Visor de Imágenes en C con SDL2

Un visualizador de imágenes ligero escrito en C.

## Requisitos
* Tener instalado C/C++ compiler (GCC, Clang o MSVC).
* Tener instalada la librería SDL2 y SDL2_image.
  * **Debian/Ubuntu:** `sudo apt install libsdl2-dev libsdl2-image-dev`
  * **Windows:** Descargar las librerías de desarrollo de libsdl.org.
